<h1> MCP4725 12 Bit DAC </h1>

<h4> MCP4725 has the following functions</h4>

```
void MCP_Write_DAC_Fast_Mode(uint16_t data);
void MCP_Write_DAC(uint16_t);
void MCP_Write_DACnEEPROM(uint16_t);
```
